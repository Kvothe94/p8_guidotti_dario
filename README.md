NL2SNL

Il progetto fornisce una API per la traduzione di requisiti espressi in linguaggio naturale
in requisiti espressi come Formal Specification Pattern.

Per utilizzare il progetto scaricare tutto il contenuto della repository e aprire la stessa
come progetto di Eclipse.
Si esegua quindi la testSuite presente nel package tests del progetto.

Il progetto è completo di documentazione javadoc nella cartella docs, nonchè degli srs e sdd nelle
rispettive cartelle.

Il documento "modifiche.txt" contiene le modifiche effettuate nel passaggio dal modello all'implementazione.

NB: Il tool necessita di una connessione internet attiva.
